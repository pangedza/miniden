# Handmade Baskets Shop

Адаптивный статический сайт интернет-магазина корзинок ручной работы.

## Состав
- **index.html** — главная с промо-блоком, рассказом о мастерской и контактами.
- **catalog.html** — каталог с карточками товаров и модальным подтверждением добавления.
- **cart.html** — корзина с изменением количества, подсчётом итоговой суммы и оформлением заказа.
- **style.css** — стилизация в тёплых оттенках, адаптивная сетка.
- **script.js** — логика корзины с сохранением в `localStorage`.
- **.github/workflows/deploy.yml** — автоматический деплой на GitHub Pages при пуше в ветку `main`.

## Запуск локально
Просто откройте `index.html` в браузере. Для корректной работы корзины и модального окна страницы должны находиться в одной директории.

## Деплой на GitHub Pages
1. Создайте репозиторий `handmade-baskets` и загрузите файлы.
2. Убедитесь, что включен GitHub Pages (Settings → Pages → Source: GitHub Actions).
3. Пуш в ветку `main` автоматически запустит workflow и опубликует сайт.

Готово! Теперь можно делиться ссылкой на витрину магазина корзинок.
