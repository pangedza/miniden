# Handmade Hello

Простая приветственная страница в стиле handmade/vintage.  
Готова для деплоя через GitHub Pages (Actions).

## Состав
- `index.html` — основная страница
- `style.css` — стили
- `.github/workflows/deploy-pages.yml` — автодеплой на Pages
- `README.md` — описание

## Локальный запуск
Открой `index.html` в браузере.
